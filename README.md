##  專案結構

```
iii-internship-team/
├── index.html              # 主要網頁檔案
├── css/style.css           # CSS
├── js/script.js            # JavaScript 功能檔案 <--在這裡加你們的個人資料、網站
├── images/                 # 圖片資料夾
└── README.md              # 說明文件
```

##  更新成員資訊

### 編輯 script.js:
```javascript
members: [
        {
            id: 1,
            name: "你的姓名",
            description: "打自我介紹",
            image: "images/member1.jpg",
            website: "網站連結/資料夾路徑"
        },]
```

### 欄位說明
- `id`: 成員編號 (1-6)
- `name`: 成員姓名
- `description`: 個人簡介
- `image`: 照片路徑
- `website`: 個人網站網址

## 📸 添加成員照片

1. 將照片放入 `images/` 資料夾
2. 建議尺寸：300x300 像素正方形
3. 建議格式：JPG 或 PNG
